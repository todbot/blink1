Welcome to Blink1Control
=================

Blink1Control is a cross-platform application for controlling 
a blink(1) USB RGB LED notification device. 

Blink1Control  is an event-based system, where events on your 
computer or on the Net (via IFTTT!) trigger color patterns on the blink(1).

Blink1Control is an open source project.  The main development
lives at https://github.com/todbot/blink1/


Installing & Running Blink1Control
----------------------------

To run Blink1Control, double-click on Blink1Control.exe

Blink1Control doesn't require an installer.
Just unzip the Blink1Control-win.zip file copy the
Blink1Control folder to a location of your choosing.

If you prefer an installer, download the Blink1Control installer
from http://blink1.thingm.com/blink1control
